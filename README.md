# CyberCert

Android app for tracking cybersecurity certifications and staying up to date with infosec news.

## Features
- Track progress across 40+ certifications from CompTIA, EC-Council, Offensive Security, ISACA, ISC2, GIAC and more
- Log study sessions and track hours per certification
- Set exam dates with countdown timers
- Live cybersecurity news feed from TheHackerNews, Krebs on Security, and SANS ISC
- Bookmark articles for later reading
- Clean Material3 UI with dark mode

## Tech Stack
Kotlin · Jetpack Compose · Material3 · Room · WorkManager · Retrofit · MVVM

## Build
Standard Android build. Requires Android Studio Hedgehog or later.
